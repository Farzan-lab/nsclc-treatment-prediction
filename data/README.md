# data/

## What is this folder?

The storage location for **all data files** in the project.

⚠️ **This folder is listed in `.gitignore` — no data will ever be pushed to GitHub.**
This is both to protect patient privacy and to avoid large file sizes in the repository.

---

## Subfolders

### `raw/` — Original Raw Data

The original MSK-CHORD files downloaded directly from cBioPortal.
**Never modify these files.**

Expected files:

```
data/raw/
├── data_clinical_patient.txt     ← Patient-level clinical information
├── data_clinical_sample.txt      ← Sample-level information
├── data_mutations.txt            ← Genomic mutation data
├── data_timeline_treatment.txt   ← Treatment history
├── data_timeline_status.txt      ← Patient status over time
└── data_timeline_pdl1.txt        ← PDL1 expression data
```

---

### `processed/` — Processed Data

The output of **Phase 1** (data preparation) lives here.

```
data/processed/
└── nsclc_final.csv    ← Final dataset, ready for modeling
```

**Specifications of `nsclc_final.csv`:**
- 6,110 NSCLC patients
- 66 feature columns (16 clinical + 50 genomic)
- 3 target columns (TREATMENT_TYPE, OS_EVENT, OS_MONTHS)
- No missing values

---

### `splits/` — Data Splits

Index files that specify which split each patient belongs to.
These files are generated by `src/data/dataset.py`.

```
data/splits/
├── test_indices.npy          ← Test set indices (fixed, created once)
├── fold_0_train_indices.npy
├── fold_0_val_indices.npy
├── fold_1_train_indices.npy
└── ...
```

**Why do we save the indices?**
To ensure that all models (baseline and cross-modal) are evaluated on
exactly the same train/val/test patients, guaranteeing a fair comparison.

## Setup Instructions (starting from scratch)

```bash
# 1. Download MSK-CHORD from cBioPortal
#    URL: https://www.cbioportal.org/study/summary?id=msk_chord_2024

# 2. Place the downloaded files in data/raw/

# 3. Run the data preparation notebook
jupyter notebook notebooks/00_data_preparation/

# 4. The output will be saved to data/processed/nsclc_final.csv
```
